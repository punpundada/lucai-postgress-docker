# lucai-postgress-docker

1.To install dependencies:

```bash
bun install
```
3.Create a .env file

```bash
cp .env .env.example
```

4.Change environment variables 

```bash
docker compose up
```

To run:

```bash
bun run dev
```

This project was created using `bun init` in bun v1.1.8. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
